# Pièces d’échecs (coalitions)

- **`ChessPiecePng.jsx`** — composant `<ChessPieceImg />` : chemins vers `public/chess/pieces/{eau|feu}/{light|dark}/{k|q|r|n|p|bg|bd}.png`.
- **Fous** : `bg` sur case **claire**, `bd` sur case **foncée**.
- **Terre / Air** : pas d’assets → repli sur le thème **feu** jusqu’à ce que tu ajoutes des dossiers `terre/` et `air/` avec les mêmes noms de fichiers.
- **Assets** : déposés **manuellement** ; voir `public/chess/pieces/README.md`.
